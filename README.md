# Japanese-mirai-robotics-corporate-site

A production-ready corporate website for **"Mirai Kaigo Robotics" (未来介護ロボティクス)**, a fictional Japanese health-tech and eldercare robotics company. 

The project features a clean, high-end "medical tech" aesthetic designed to convey safety, trust, and advanced technology. It utilizes modern web technologies to create a premium, interactive user experience.

## Key Features
- 🚀 **Next.js 14 App Router** architecture for static generation and routing
- 🎨 **Premium Japanese Healthcare Design System** using Tailwind CSS (Mist Blue, Soft Teal, Navy)
- 🪄 **GSAP & ScrollTrigger** for buttery-smooth reveal animations on scroll
- 🤖 **Interactive 3D WebGL** elements using React Three Fiber and Three.js (Interactive glowing AI core & particle backgrounds)
- 📱 **Fully Responsive**, mobile-first design
- 🇯🇵 **Japanese-First Content** tailored for the domestic B2B healthcare market

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- **Framework:** Next.js
- **Styling:** Tailwind CSS
- **Animations:** GSAP (GreenSock)
- **3D / WebGL:** Three.js, React Three Fiber, React Three Drei
- **Icons:** Lucide React
